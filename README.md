# Markdown

- ### An intro to Markdown
Markdown is such a useful and fun way of making descriptions — and writing text in general — more alive and less boring by making formatted text far easier. This guide will go through the basics for beginners and some other cool and more advanced things you didn't know were possible.

This guide will go through GitHub Markdown's basics and more advanced mechanics, while making it easier for you to understand as the current official guide isn't very up to date, lacks info and doesn't even work sometimes.

- ### HTML's little brother
Markdown is an easy to use markup based on HTML, which means that you can use *most* of HTML's anchor tags such as **links** and **images**.

- ### You might already know about Markdown
Markwon is often used for **blogs** or **posts** and **articles** on social media. For example, Wikipedia is written in a slightly modified version of Markdown and Reddit users use it for writing their posts.  
Even this *Readme* is written in Markdown, although it is a **modified version built for GitHub** which is what you will learn about in this guide.
