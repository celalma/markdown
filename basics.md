# Markdown basics

## Intrduction
Here is a quick guide on how to do basic writing and using formatting syntax.
Before anything, you need to create an `.md` file either by creating one on GiHUb and writing there directly or by using another code editor or program to write in (you can use *Notepad*, but it is not recommanded...)
For beginners, we advise creating a new file in your **repository** and giving it a name like `README.md`.

## Basic Syntax

To create a **heading**, use `#` before your title with a space between. The more `#` you use, the less important the text will be. You can use between 1 and six `#` to create a heading.


In markdown :
```
# Heading1

## Heading 2

### Heading 3

...
```
Formatted :

# Heading 1
## Heading 2
### Heading 3 

You can use different symbols to change the style of your text :

| Style | Plain text | Result (formatted) |
| --- | --- | --- |
| Italic | \*text* | *text* |
| Italic | \_text_ | _text_ |
| Bold | \**text** | **text** |
| Subscript | \<sub>text</sub> | <sub>text</sub> |
| superscript | \<sup>test</sup> | <sup>text</sub> |

Note that you can use a mix of them to use <sub>multiple</sub> ***styles***.
